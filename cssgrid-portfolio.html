Jeg havde fået til opgave at arbejde med CSS grid. Jeg har udarbejdet et layout ved hjælp af CSS grid, der kan benyttes til videre arbejde. Det er derfor ikke en færdig hjemmeside.

Ved hjælp af CSS grid, har jeg bygget et layout der indeholder en navigation, header i form af et billede, main content i form af artikler og en footer. Jeg har arbejdet med at bruge grid inde i grid. Det vil sige jeg har bygget grid layouts inde i main, samt i navigationen. Html og CSS kan ses i GitHub link øverst.

Står du og mangler en med erfaring i CSS Grid? Så klik på knappen under, og kontakt mig.
